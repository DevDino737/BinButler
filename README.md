# BinButler

BinButler is a local curbside trash bin service website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Online signup form
- Pickup request form
- FAQ section
- Pricing cards
- Google Apps Script integration

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
